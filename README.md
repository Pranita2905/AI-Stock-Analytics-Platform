# 📈 AI Stock Analytics Platform

 Live Application: [Click Here to Launch AI Stock Analytics Platform](https://ebxbjhyf6cpdq2fjqxvzkh.streamlit.app/)

An advanced Machine Learning-powered Stock Market Analytics and Prediction platform built using Python, Streamlit, Scikit-learn, Plotly, and real-time stock APIs.
This project predicts stock closing prices using historical market data and provides a professional analytics dashboard with interactive visualizations and AI-powered trading signals.

# ⚙️ Installation
## 1️⃣ Clone Repository
```bash
git clone https://github.com/Pranita2905/AI-Stock-Analytics-Platform.git
```
---
## 2️⃣ Move to Project Folder
```bash
cd AI-Stock-Analytics-Platform
```
---
## 3️⃣ Create Virtual Environment
```bash
python -m venv .venv
```
---
## 4️⃣ Activate Virtual Environment
### Windows
```bash
.\.venv\Scripts\activate
```
### Linux / Mac
```bash
source .venv/bin/activate
```
---
## 5️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```
---
# ▶️ Run the Application
```bash
streamlit run app_streamlit.py
```
